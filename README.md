# Monotone Convergence Theorem — Visual Demo

**Educational Python visualization** showing two **perfectly straight** monotone sequences converging to the same limit (1.0).

- Left plot: **Increasing** sequence (starts at 0.25 → 1.0)
- Right plot: **Decreasing** sequence (starts at 1.75 → 1.0)

Both are **uniform/linear** (straight slanted lines) and **strictly monotone**, illustrating the Monotone Convergence Theorem perfectly.

## Features
- 100% **pure Python** — no NumPy required
- Only needs `matplotlib` (easy to install)
- High-resolution PNG output
- Clean, publication-ready plots
- Fully commented code
